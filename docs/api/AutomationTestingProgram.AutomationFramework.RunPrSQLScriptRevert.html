Class RunPrSQLScriptRevert

This test step to run a sql script from paramterized values.Only works for PR applications. This one of the more complicated test step as it relys on the test step to be a databaseData type object. note the following format define formCode = 'VARIABLEOVERWRITE1' define formYear = VARIABLEOVERWRITE2 define orgCode = 'VARIABLEOVERWRITE3' define dateOfExecution = 'VARIABLEOVERWRITE4'.
